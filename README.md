# Film Effect Image Processing Website

This project provides a simple web-based interface to apply film-style filters and basic image adjustments. Users can upload images via drag-and-drop or file selection, preview changes in real-time, and export the result in various formats.

## Features
- **Film Filters**: Vintage, Cinema and Noir presets.
- **Adjustments**: Brightness, contrast and saturation sliders.
- **Multiple Uploads**: Drag-and-drop area and file input.
- **Real-time Preview**: Changes are immediately rendered on a canvas.
- **Export Options**: Select format (PNG, JPEG, WebP), quality and scaling before download.

Open `index.html` in a web browser to start editing.
